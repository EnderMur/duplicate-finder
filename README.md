# duplicate-finder
Простая CLI-утилита на Rust для поиска дубликатов файлов по SHA-256 хешу с возможностью удалить их или переместить в отдельную папку.

---
## Установка

### Клонирование репозитория

```bash
git clone https://github.com/EnderMur/duplicate-finder.git
cd duplicate-finder
```

### Сборка проекта

Убедитесь, что у вас установлен Rust и Cargo.

```bash
cargo build --release
```

Готовый бинарный файл будет находиться в:

```bash
target/release/duplicate-finder
```

## Использование

### Аргументы

#### `-p, --path <DIR>`

Путь к директории, в которой нужно искать дубликаты.

- Если аргумент передан, программа сразу начинает сканирование этой папки.
- Если аргумент не передан, программа попросит ввести путь вручную после запуска.

Пример:

```bash
cargo run -- --path "C:/Users/User/Documents"
```

#### `--move-to <DIR>`

Папка, в которую будут перемещаться дубликаты, если в меню выбрать перенос вместо удаления.

- Если аргумент передан, дубликаты будут перемещаться именно в эту директорию.
- Если аргумент не передан, используется папка `.duplicates` внутри сканируемой директории.
- Аргумент не включает перенос автоматически: после поиска программа всё равно спросит, что делать с найденными дубликатами.

Пример:

```bash
cargo run -- --path "C:/Users/User/Documents" --move-to "D:/Temp/duplicates"
```

#### `-h, --help`

Показывает краткую справку по аргументам и примерам запуска.

Пример:

```bash
cargo run -- --help
```

#### `-V, --version`

Показывает версию программы.

Пример:

```bash
cargo run -- --version
```

### Запуск с указанием пути

```bash
cargo run -- --path "C:/Users/User/Documents"
```

или

```bash
./duplicate-finder --path "/home/user/files"
```

### Перемещение дубликатов вместо удаления

По умолчанию при выборе перемещения дубликаты попадут в папку `.duplicates` внутри указанного пути.

```bash
cargo run -- --path "C:/Users/User/Documents"
```

Если нужна своя папка назначения:

```bash
cargo run -- --path "C:/Users/User/Documents" --move-to "D:/Temp/duplicates"
```

### Запуск без аргументов

```bash
cargo run
```

После запуска программа попросит вручную ввести путь.
Во время работы показывается простой прогресс-бар для сканирования и хеширования файлов.

### Что происходит после поиска

После того как программа найдёт дубликаты, она покажет интерактивное меню:

- `0` - ничего не делать и выйти
- `d` - удалить дубликаты во всех группах, оставив первый файл
- `m` - переместить дубликаты во всех группах, оставив первый файл
- `1` - оставить первый файл в каждой группе и удалить остальные
- `i` - пройти по группам по одной и выбрать действие отдельно
